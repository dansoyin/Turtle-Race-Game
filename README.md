# Turtle-Race-Game
A simple interactive racing game built using Python's Turtle Graphics.

🚀 How to Run
Make sure you have Python installed on your system.
Download the main.py file.
Run the script using your terminal or IDE:

Bash'''
python main.py
'''
🎮 How to Play
A pop-up window will appear asking for your bet.
Type one of the colors: red, orange, yellow, green, blue, or purple.
Watch the race!
Check the terminal to see if you won or lost.
